### Option 1: Download APK (Recommended)

1. Download the latest APK from the [Releases](../../releases) page
2. Enable "Install from Unknown Sources" in your Android settings
3. Open the APK file and tap "Install"
4. Launch WelderTools from your app drawer

## Using the App

### Main Menu

When you launch WelderTools, you'll see the main menu with these options:

1. **MIG Welding Settings**
2. **TIG Welding Settings**
3. **Arc/Stick Welding Settings**
4. **Wire Speed Calculator**
5. **Machine Information**
6. **Material Properties**
7. **Hours & Parts Tracking**

### Getting Welding Settings

1. **Tap** the welding type you want (MIG, TIG, or Arc)
2. **Select** your material from the dropdown:
   - Mild Steel
   - Stainless Steel
   - Aluminum
3. **Select** your material thickness:
   - 1/16"
   - 1/8"
   - 3/16"
   - 1/4"
4. **Tap** "Get Settings" button
5. **Read** the recommended settings:
   - Voltage/Amperage
   - Wire speed or electrode
   - Gas type and flow
   - Travel speed
   - And more!

### Wire Speed Calculator

1. **Tap** "Wire Speed Calculator" from main menu
2. **Select** material type
3. **Select** wire size (0.023" - 0.045")
4. **Select** thickness category (thin/medium/thick)
5. **Tap** "Calculate"
6. **View** detailed wire speed recommendations

### Machine Information

1. **Tap** "Machine Information" from main menu
2. **Select** a brand from the dropdown:
   - Miller
   - Lincoln
   - ESAB
   - Hobart
   - Everlast
   - Fronius
   - All (shows all brands)
3. **Browse** machine specifications and features

### Material Properties

1. **Tap** "Material Properties" from main menu
2. **Select** a material:
   - Mild Steel
   - Stainless Steel
   - Aluminum
   - Cast Iron
3. **Learn** about composition and weldability

### Tracking Your Work

Keep track of your welding hours and parts made:

1. **Tap** "Hours & Parts Tracking" from main menu
2. **Enter** hours worked (e.g., 4.5)
3. **Enter** parts made (e.g., 12)
4. **Optionally** add a description
5. **Tap** "Log Entry" to save
6. **Tap** "View Stats" to see:
   - Total hours worked
   - Total parts made
   - Recent entries
7. **Tap** "Clear Log" to start fresh (use with caution!)

## Troubleshooting

### App Won't Install

- Make sure "Install from Unknown Sources" is enabled
- Check that you have Android 5.0 (Lollipop) or newer
- Free up storage space if needed

### App Crashes

- Try clearing the app's cache and data
- Reinstall the app
- Check that you're running Android 5.0+

### Tracking Not Saving

- Grant storage permissions in Android settings
- Check that you have available storage space
- Try clearing and starting fresh

### Settings Not Showing

- Make sure you selected both material AND thickness
- Tap "Get Settings" button after selecting options
- Try a different material/thickness combination
